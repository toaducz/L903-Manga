# Project Overview: L903-Manga

## Description
L903-Manga is a modern, high-performance manga reader application built with Next.js 15. It aims to provide a premium reading experience with a focus on speed, aesthetics, and user-friendly features. The project is designed to be extensible and potentially integrated with platforms like Zalo (using `zmp-ui`).

## Main Goals
- **Seamless Reading Experience**: Fast loading times and smooth transitions.
- **Premium UI/UX**: Utilizing `framer-motion` for animations and a curated design system.
- **Robust State Management**: Using `Recoil` for global state and `React Query` for data fetching.
- **Modular Architecture**: Organized codebase to ensure maintainability and scalability.

## Target Audience
Manga enthusiasts looking for a clean, feature-rich, and fast reading platform.
