# Features & Roadmap

## Current Features
- **Manga Discovery**: Browse by categories, popular, and latest updates.
- **Advanced Search**: Filter and find manga by genre, status, and authors.
- **Manga Reader**: High-performance reader supporting multiple modes (vertical/horizontal).
- **Reading History**: Automatically tracks progress and allows resuming from where the user left off.
- **Manga Details**: Comprehensive info, including synopsis, chapter list, and author details.
- **Mobile Optimized**: Fully responsive design with a focus on mobile usability.

## Planned Features / Roadmap
- **User Accounts**: Profile management, bookmarks, and personalized recommendations.
- **Offline Reading**: Support for downloading chapters for offline access (PWA features).
- **Social Integration**: Comments, ratings, and sharing manga with friends.
- **Advanced Settings**: Highly customizable reader (brightness, background color, transition effects).
- **Multi-language Support**: Interface and content in multiple languages.
- **Zalo Integration**: Full support for Zalo Mini Program environment.
